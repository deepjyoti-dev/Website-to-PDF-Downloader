# Website-to-PDF-Downloader
A modern Python GUI application that converts any website into a PDF document using wkhtmltopdf, pdfkit, and Tkinter.
Features
Convert websites into PDF
Simple and clean GUI
Save PDF anywhere on system
Background threading (UI does not freeze)
URL validation
Status updates
Error handling
A4 PDF formatting
UTF-8 support
Technologies Used
Python
Tkinter
pdfkit
wkhtmltopdf
Multithreading
Requirements

Install Python packages:

pip install pdfkit

Install wkhtmltopdf:

Windows

Download from:

wkhtmltopdf Official Website

Default installation path:

C:\Program Files\wkhtmltopdf\bin\wkhtmltopdf.exe
How to Run
python app.py
